# Кластеризация научных статей по спортивному питанию
Данный проект представляет собой интерактивное веб-приложение на базе Streamlit для анализа и визуализации кластеризации научных публикаций в области спортивного питания. В основе анализа лежит тематическое моделирование (LDA) и визуализация с помощью t-SNE.

## Цель проекта
Помочь исследователям, студентам и всем интересующимся быстро ориентироваться в массиве научных статей по спортивному питанию: находить статьи по темам, фильтровать по авторам и годам, изучать динамику публикаций, а также получать рекомендации похожих материалов.

## Описание данных
Исходные данные загружаются из CSV-файла LDA4_final.csv. Файл должен содержать следующие колонки:

title – заголовок статьи

authors – список авторов

journal – название журнала

clean_abstract – очищенный текст аннотации

main_topic_name – название основной темы (назначено по результатам LDA)

main_topic_probability – вероятность отнесения к основной теме

## Используемые методы
Тематическое моделирование (LDA) – для выделения пяти ключевых тем в корпусе статей.

t‑SNE – для снижения размерности векторов вероятностей тем до двумерного пространства с целью визуализации кластеров.

Фильтрация и агрегация – реализованы с помощью pandas.

Визуализация – библиотека Plotly Express для интерактивных графиков.

## Функциональность приложения
### Боковая панель (фильтры)
Выбор авторов (из общего списка).

Выбор тем (чекбоксы по каждой теме).

Диапазон годов публикации.

Диапазон объёма статьи (количество слов).

Кнопка сброса всех фильтров.

### Основная страница
Ключевые метрики – общее число статей, количество тем в выборке, выбранный диапазон лет, средняя длина статьи.

Интерактивная карта статей (t‑SNE)

Каждая точка – статья.

Цвет соответствует основной теме.

При наведении отображается заголовок, авторы, тема, вероятность, год, объём.

Легенда тем расположена справа.

Статистика по темам

Горизонтальная столбчатая диаграмма – распределение статей по темам.

Динамика публикаций по годам (площадная диаграмма).

Детальный анализ темы

Выбор темы из выпадающего списка.

Отображение интерпретации темы, количества статей, диапазона лет.

Таблица со статьями (название, авторы, год, журнал, вероятность).

Возможность скачать данные по выбранной теме в CSV.

### Поиск и рекомендации

Вкладка «Поиск» – полнотекстовый поиск по аннотациям и заголовкам, сортировка результатов, пагинация.

Вкладка «Похожие статьи» – выбор статьи-образца, поиск пяти ближайших статей в пространстве t‑SNE (евклидово расстояние) с оценкой сходства в процентах.

## Установка и запуск
Клонируйте репозиторий:
```
git clone https://github.com/your-repo/sports-nutrition-clustering.git
cd sports-nutrition-clustering
```
Создайте виртуальное окружение (рекомендуется) и активируйте его:
```
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
```
Установите зависимости:

```
pip install -r requirements.txt
```
Поместите файл с данными LDA4_final.csv в корневую папку проекта (или укажите правильный путь в коде, если файл лежит в другом месте).

Запустите приложение:

```
streamlit run app.py
```
